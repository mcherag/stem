Description

Learn to build any app & Start your own app based business as a freelance iOS developer.

After taking this course, students will completely understand difficult concepts:

  • Apple's latest UI Framework - SwiftUI
  • Create apps that use Machine Learning using Apple’s new CoreML
  • Augmented Reality apps using Apple’s new ARKit
  • How to wireframe, mockup and prototype any app idea
